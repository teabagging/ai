Greedy Removal with Stack: Iterate through each digit of the input string. Use a stack to keep track of the digits of the result. For each digit, if it is smaller than the top of the stack and we still have digits left to remove (`k > 0`), pop the stack to remove the larger digit. This ensures that we remove larger digits as early as possible to minimize the result.2. Handle Remaining Removals: After processing all digits, if there are still digits left to remove (`k > 0`), remove them from the end of the stack, as the remaining digits will be in non-decreasing order.3.
